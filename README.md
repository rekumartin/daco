# Classic Snake Online

A classic-vibe Snake game made with HTML5 canvas and vanilla JavaScript.

## Run locally

```bash
npm start
```

The server binds to `0.0.0.0:8080` by default so you can expose it publicly (for example via a cloud VM, reverse proxy, or tunnel).

Environment variables:
- `PORT` (default `8080`)
- `HOST` (default `0.0.0.0`)

Then open:

- Local: `http://localhost:8080`
- LAN/Public: `http://<your-server-ip>:8080`

## Controls

- `Enter`: Start / restart
- `Arrow keys` or `WASD`: Move
- `Space`: Pause / resume
